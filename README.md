Minecraft Glyph Matrix Gallery

A custom Android application designed to bring Minecraft pixel art directly to the back of your Nothing Phone using the Glyph Matrix SDK.

Features
Massive Pixel Art Library:** Browse through carefully mapped designs of Minecraft Mobs, Items, Food, and Blocks.
Multi-Device Support:** Custom-scaled matrix arrays for both the standard 25x25 grid (Phone 3) and the zoomed 17x17 grid (Phone 4a Pro).

Notification Export: Save any Minecraft design directly to your phone's notification settings to use as a custom alert.

You do not need to build this from source to use it! 
1. Go to the **[Releases](../../releases)** page on the right side of this repository.
2. Download the latest `.apk` file to your Android device.
3. Open the file to install the app. *(Note: You may need to allow "Install from unknown sources" in your Android settings).*

Developer Notes
Built with Kotlin and Jetpack Compose. This project utilizes custom mathematical UI rendering to simulate the physical gaps and corner-rounding of real hardware LEDs.
